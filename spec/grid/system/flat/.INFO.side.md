# Désignation des côtés

                    North
                   -------
                 /         \
    North East  /           \   North West
               /             \
               \             /
    South East  \           /   South West
                 \         /
                   -------
                    South

# Ordre de récupération
L'ordre de récupération se fait dans un mouvement anti-horaire,
à partir du sud-ouest.

    side = {
      SOUTH_WEST: 0
      NORTH_WEST: 1
      NORTH:      2
      NORTH_EAST: 3
      SOUTH_EAST: 4
      SOUTH:      5
    }
